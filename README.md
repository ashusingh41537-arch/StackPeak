# 💼 StackPeak - Job Portal Web Application

StackPeak is a Full Stack Job Portal Web Application that connects job seekers with employers. Users can browse job listings, apply for jobs, and manage their profiles, while recruiters can post and manage job openings.

This project is built using modern technologies to demonstrate full stack development skills and real-world application architecture.

---

## 🚀 Features

### 👤 Job Seeker Features

* User Registration & Login
* Browse Available Jobs
* Apply for Jobs
* Manage Profile

### 🏢 Recruiter Features

* Post Job Listings
* Update / Delete Jobs
* View Applicants

## ⚙️ Installation & Setup

### 1️⃣ Clone Repository

```bash id="3a45qo"
git clone https://github.com/ashusingh41537-arch/StackPeak.git
cd StackPeak
```

---

### 2️⃣ Backend Setup

```bash id="w71n5q"
cd backend
mvn spring-boot:run
```

Backend runs on: http://localhost:8080

---

### 3️⃣ Frontend Setup

```bash id="dr1svx"
cd frontend
npm install
npm start
```

Frontend runs on: http://localhost:3000

## 🔗 API Endpoints (Sample)

| Method | Endpoint   | Description     |

| GET    | /jobs      | Get all jobs    |

| DELETE | /jobs/{id} | Delete a job    |
