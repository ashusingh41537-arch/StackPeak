# 🚀 StackPeak - Job Portal Full Stack Application

StackPeak is a Full Stack Web Application designed as a Job Portal where users can explore job opportunities, apply for jobs, and companies can post job listings.

---

## 🔹 Features

* 🔐 User Authentication (Login & Register)
* 🧑‍💼 User & Company Management
* 📄 Job Posting & Listing
* 📬 Apply for Jobs
* 🔎 Search & View Jobs
* 🔒 Secure Backend using JWT Authentication
* 🔄 REST API Integration

## 🔹 How to Run the Project

### 🔸 Backend (Spring Boot)

1. Open backend folder in IDE (IntelliJ / Eclipse)
2. Configure MySQL database in `application.properties`
3. Run the Spring Boot application

### 🔸 Frontend (React)

1. Open frontend folder in terminal
2. Install dependencies:

   ```
   npm install
   ```
3. Start the app:

   ```
   npm start
   ```
